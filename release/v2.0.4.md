# Zorg MemoryDB v2.0.4

This release adds the verified Android install path for LAN Command Chat.

- Replaces the web chat's Compact control with an Android app download link.
- Publishes the Android client APK as a release asset.
- Bumps the Android application version to 2.0.4.
- Preserves the native chat, local/remote routing, gauges, Context window
  telemetry, and Memory Brain 3D toggle.

The release contains no private scheduler credentials, signing keys, local SDK
state, or operator-only paths.
