# Zorg MemoryDB v2.0.3

This corrective full release completes the native Android LAN Command Chat
surface.

- Native chat window, history, send states, and local/remote route fallback.
- Native Queries/sec, Cache hit, Writes/sec, DB size, and Context window
  readouts.
- Memory Brain 3D toggle remains available through the existing supported
  surface.
- Canonical fixed-seed image generation remains part of the `zorg-db-memory`
  skill.

The release contains no private scheduler credentials, signing keys, local SDK
state, generated APKs, or operator-only paths.
