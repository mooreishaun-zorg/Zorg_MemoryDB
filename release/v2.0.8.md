# Zorg MemoryDB v2.0.8

This full point-in-time release aligns the Android client, LAN Command Chat,
and the public `zorg-db-memory` package after the verified Android 2.0.7 run.

## Included

- Android client loads the canonical responsive LAN Command Chat `/chat` page.
- Android uses the variable-driven LAN profile route and `theme=system` so the
  phone's light/dark setting controls the rendered surface.
- Mobile scrolling keeps the lower live gauge panel reachable, including the
  `Memory 3D`/`Gauges` toggle and connected Memory Brain 3D view.
- LAN Command Chat preserves live status, context-window telemetry, gauges,
  profiles, history, send/loading/error/retry, refresh, and Android install
  link behavior.
- The package retains DB-first recall, repair-first fail-closed behavior,
  runtime-generated request timing summaries, and screenshot inspection gates.

## Verification

- Android debug APK builds with the repository's Android toolchain.
- APK installs on the USB Motorola target `ZY322WGVPN` at `1080x2160`.
- The installed app reaches the real LAN Command Chat login surface.
- Existing reviewed mobile light/dark and Memory 3D toggle screenshots remain
  additive public assets; no desktop screenshot substitutes for mobile proof.
- PostgreSQL/Zorg MemoryDB tables and speed checks pass.
