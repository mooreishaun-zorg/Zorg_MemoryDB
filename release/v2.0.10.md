# Zorg MemoryDB v2.0.10

This release publishes the browser LAN Command Chat gauge update with aligned
version metadata.

## Included

- Preserves the live `Context window` gauge.
- Displays `v2.0.10` immediately before `DB size` inside the gauge tile.
- Aligns the root package and browser application metadata at `2.0.10`.

## Protected

- Android, Memory Brain 3D, backend/API, database, installer, runtime,
  browser theme behavior, and unrelated gauges are unchanged.
