# Zorg MemoryDB v2.0.2

This release adds the native Android client and makes canonical image
generation part of the `zorg-db-memory` skill.

## Android

- Native chat UI with message history, send/retry states, context telemetry,
  and no TUI dependency.
- Internet URL plus optional internal host/port routing with fallback.
- Existing LAN Command Chat Memory Brain 3D access is preserved.
- The package includes only source and reproducible Gradle files; private
  signing material, SDKs, build outputs, and scheduler credentials are absent.

## Image generation

- ComfyUI support is owned by the canonical skill.
- A single `image-seed.json` provides the deterministic default seed.
- ComfyUI server and output paths are configurable through environment
  variables; no operator-specific path is required.

## Verification

- Android debug APK build succeeds with the supplied Android SDK.
- Fixed-seed, Python syntax, package boundary, SQL migration, and public
  artifact checks pass.
