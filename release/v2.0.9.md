# Zorg MemoryDB v2.0.9

This release replaces the Android web pass-through with a native LAN Command
Chat client.

## Included

- Native Android conversation history, composer, send state, and error state.
- Native LAN Chat password login with persisted signed-cookie state; no
  password is embedded in the APK.
- Native System/Light/Dark appearance selection; System follows Android's live
  phone configuration.
- Live database Queries/sec, Cache hit, Writes/sec, Database size, health, and
  memory gauges from `/api/db/status`.
- Native Memory 3D rendering driven by `/api/graph` data from the Memory 3D
  service, with an explicit empty-graph state and a working return control.
- No Android WebView and no Android OpenClaw TUI dependency.
- The installer prepares and enables the separate `zorg-memory-3d` service on
  port `8097`, with bounded PostgreSQL queries and documented health/graph
  probes.
- Browser LAN Chat remains the owner of browser theme controls and the APK
  download link; the native Android app does not include that link.
